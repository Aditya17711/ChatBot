ChatBot

A simple rule-based chatbot built with plain HTML, CSS, and JavaScript. This project is designed for learning and quick prototyping.

Overview

  • No frameworks or build tools required.
  • Runs entirely in the browser.
  • Uses a predefined phrase-to-response mapping.
  • Good starter project for beginners.

Project Structure

ChatBot/
├── index.html   # README-style project description page
├── styles.css   # Chat interface styles
└── script.js    # Chatbot behavior and response logic

How It Works

  1. User enters a message in the input box.
  2. The send() function displays the user message.
  3. The response() function checks known phrases.
  4. The bot reply is shown in the chat area.

Sample Response Map

const responses = {
  "hi": "Hi",
  "hello": "Hi, how can I help you today!",
  "how are you": "I am fine, thanks for asking!",
  "what's your name": "My name is Lucifer.",
  "what's your age": "I am a coded chatbot, so I don't have an age.",
  "bye": "Goodbye! Have a great day!"
};

Example Conversation

User Input Bot Reply
hello Hi, how can I help you today!
what's your name My name is Lucifer.
bye Goodbye! Have a great day!

Run Locally

# Option 1: Open directly
Open index.html in your browser

# Option 2: Use a local server
python -m http.server 8000
# then open http://localhost:8000
