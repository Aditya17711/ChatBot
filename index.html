Repository Overview

ChatBot: Rule-Based Web Chat Demo

This repository is a lightweight browser chatbot project built with plain HTML, CSS, and JavaScript. It is designed for learning and quick prototyping: no build tools, no backend, and no dependencies.

Frontend-only No framework Static hosting ready Beginner friendly

Detailed Description

Purpose

  • Demonstrate chatbot UX basics with minimal code.
  • Show a simple message flow from user input to bot response.
  • Serve as a starter template for custom intent mapping.

Core Workflow

  • send() reads and validates the input.
  • show() renders messages in the chat window.
  • response() resolves a reply from predefined phrases.
