Repository Summary

Simple Rule-Based ChatBot (HTML + CSS + JavaScript)

This repository demonstrates a beginner-friendly chatbot interface that runs fully in the browser. The bot responds to predefined phrases using a lightweight JavaScript dictionary, making it easy to understand, extend, and deploy as a static site.

What This Repository Includes

  • UI Layer: Chat container, message bubbles, and input controls for user interaction.
  • Logic Layer: A response() function that maps known user phrases to bot replies.
  • Rendering: A show() helper that appends messages into the chat window.
  • Flow: A send() function to capture input, render user text, and produce a delayed bot response.
